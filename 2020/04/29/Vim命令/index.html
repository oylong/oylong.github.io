Vim命令

发表于 2020-04-29 | 分类于 Linux
没什么简介可以说

Vim

命令名称 解释
set nn 显示行号
gg 跳转到文件开头
/ 向前/后搜索
n/N 查找上/下一处
0(零)/$ 光标所在行行首/行末
{ | } 段首/尾
- / + 前/后一行行首
() 句首
) 下一句首
M 屏幕中间行
L 屏幕所显示的底行
hjkl 左下上右
x 删除光标所在字符
R 替换模式（可以替换任意字符）
r 单个替换
dd 删除光标所在的行
D 删除至行末（从光标位置开始）
s 删除字符并插入（单个字符删除，并进入插入模式）
S 删除行并插入（整行删除）
| >> /<<| 缩进（相当于一个tab）/反缩进
= 自动格式化
J 合并上下两行
I 插入到行首
i 插入
a 在光标后附件或追加
A 在行末追加
P/p 粘贴（后）/粘贴（前）
ZZ 保存退出编辑(vi，含保存)
ZQ 不保存退出编辑
